Aulas Introdutórias

O que é HTML?

HTML é a linguagem de marcação usada para estruturar conteúdos na web. Ele define títulos, parágrafos, links e muito mais.

Exemplo de Código:


<!DOCTYPE html>
<html>
  <head>
    <title>Meu primeiro site</title>
  </head>
  <body>
    <h1>Bem-vindo ao meu site!</h1>
    <p>Este é um parágrafo de texto.</p>
  </body>
</html>

O que é CSS?

CSS é o que usamos para deixar os sites bonitos: cores, tamanhos, posicionamentos e estilos.

Exemplo de Código:


/* CSS para mudar a cor do fundo */
body {
  background-color: #f0f0f0;
}

/* CSS para centralizar um título */
h1 {
  text-align: center;
}

O que é JavaScript?

JavaScript dá vida ao seu site. Com ele, podemos fazer animações, interações e lógica.

Exemplo de Código:


<button onclick="alert('Olá, mundo!')">Clique Aqui</button>

Exemplo Completo (HTML, CSS e JS)

Agora, vamos juntar tudo o que aprendemos:


<!DOCTYPE html>
<html>
  <head>
    <title>Meu site interativo</title>
    <style>
      body {
        background-color: #e8f5e9;
        text-align: center;
      }
      button {
        padding: 10px;
        background-color: #4CAF50;
        color: white;
        border: none;
        border-radius: 5px;
        cursor: pointer;
      }
      button:hover {
        background-color: #45a049;
      }
    </style>
  </head>
  <body>
    <h1>Bem-vindo ao meu site interativo!</h1>
    <button onclick="alert('Você clicou no botão!')">Clique Aqui</button>
  </body>
</html>
